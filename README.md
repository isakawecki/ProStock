# 📦 ProStock — Sistema de Gerenciamento de Estoque

## 📖 Sobre o Projeto

O **ProStock** é um sistema web full stack desenvolvido para auxiliar **pequenas empresas** no gerenciamento de estoque de forma prática, moderna e eficiente.

O sistema centraliza operações essenciais do negócio, permitindo controlar **produtos, fornecedores, compras, vendas, funcionários e relatórios** em uma única plataforma.

Seu design foi desenvolvido com foco em **adaptabilidade de identidade visual (branding)**, permitindo fácil personalização para diferentes empresas, marcas, cores e estilos visuais.

---

## ✨ Funcionalidades

* Login seguro de usuários
* Controle de acesso administrativo
* Cadastro e gerenciamento de produtos
* Controle de estoque
* Cadastro de fornecedores
* Registro de compras
* Registro de vendas
* Gestão de funcionários
* Dashboard com estatísticas e indicadores
* Relatórios do sistema
* Tema claro e escuro
* Configurações de conta
